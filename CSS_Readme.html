SCSS file descriptions

[base] folder

  • _fonts.scss

    This is where external fonts get loaded.

  • _global.scss

    This is where I put the CSS that is being/can be used by any page. Like responsive layout container, or body tag.

  • _mixins.scss

    This is where the SCSS mixins reside.

  • _reset.scss

    This resets predefined browser styles.

[basics] folder

  • _button.scss

    This is where I keep the styles of the buttons that are being used on multiple pages. Like the messages and more icons in the header.

  • _links.scss

    This is currently unused.

  • _typography.scss

    This is currently unused.

[pages] folder

  • _page_admin.scss

    Enything specific to Page Admin page(s) resides here.

  • _page_user_screen.scss

    Enything specific to User page(s) resides here.

[plugins] folder

This folder is currently unused.

[shared_content] folder

This folder is currently unused.

main.scss

This is the file that imports everything and generates the CSS file
